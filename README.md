# AssistantHelper
 a py program for Fat Xu :)

## Guide

测试了Microsoft Office，没有测试WPS

1. 使用pip安装 pandas，pypiwin32，openpyxl

    ```
    pip install pandas
    pip install pypiwin32
    pip install openpyxl
    ```

2. 将每一题存在 `题号.docx` 文档里，在每个文档中用换行留好空位

3. `data.docx` 存储学生信息

    样例：

    | name | score | index |
    |------|-------|-------|
    | sb   | 0     | 1,2,3 |
    | cxk  | 50    | 1,2   |
    | FX   | 100   | 全对    | 

4. 运行py文件

5. 等待片刻，查看结果
